# 社区行为准则

## 我们的承诺

我们致力于为所有人提供一个欢迎和鼓励的社区。我们承诺SteamAudioDLL社区的每个参与者都将受到尊重和尊严的对待。

## 我们的标准

有助于创建积极环境的行为示例包括：

- 使用欢迎和包容的语言
- 尊重不同的观点、观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同情
- 帮助他人学习和成长

不可接受的行为示例包括：

- 使用性化的语言或图像
- 骚扰、侮辱/贬低性评论和人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 在专业环境中可能被合理认为不适当的其他行为

## 执行

可以通过联系项目团队来报告辱骂、骚扰或其他不可接受的行为实例。所有投诉将被迅速公平地审查和调查。

所有社区领导者都有义务尊重任何事件报告者的隐私和安全。

## 范围

本行为准则适用于所有社区空间，也适用于个人在公共场所正式代表社区时。
